# Quixel in Blender

Full support of Quixel Megascans and Megaplants —— Right inside Blender.

---

Once upon the time, Quixel Bridge was the go-to solution for importing Megascans assets. It integrates seamlessly with Blender, and it was a powerful, intuitive tool for asset lookup and management.

Those good ways were gone when Quixel Bridge was discontinued and Fab took over. To this day, Fab has not provided proper integration with Blender, and despite efforts, it still lacks the smooth seach-and-browse experience that Bridge offered.

This addon, based on the now archived [Fab to Blender](https://github.com/cgmaterial/fab-to-blender), aims to recreate the Bridge experience, right inside Blender. We are working hard to make it as native and easy-to-use as possible, to make Quixel integration seamless and enjoyable for Blender users.

We are actively working on this project, and we welcome contributions from the community. If you have any ideas, suggestions, or improvements, please feel free to submit a pull request or open an issue. Stayed tuned for updates and new features!
